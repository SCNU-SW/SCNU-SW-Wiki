## 为什么要在 Github 上编辑

-   Github方便版本管理，清晰对比版本之间的区别，并且能够快速回滚。
-   强大的多人合作平台，多人提交以及多管理员审核。
-   成熟的自动化部署方案，通过审核后可自动构建整站。

<script>

	{
